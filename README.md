# [www.ryanmcg.com][www]

An incise based project for generating [my website][www].

## Usage

For help:

```bash
lein run -h
```

Run in development mode (i.e. a server that reloads source files with each
request).
```bash
lein run
```

Generate output files (e.g. html and assets).
```bash
lein run -1
```
